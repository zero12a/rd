# cron 데몬 설치 및 컨테이너 재시작시 자동 start 되게 하기
1) 컨테이너 진입 
 - docker exec -it nginx bash
2) crontab 설치
 - yum install cronie
3) crontab 자동시작되게 시작명령어 추가
 - entrypoint "/start.sh" 편집
 - "/usr/sbin/crond" 라인 추가
4) 컨테이너 나가서 컨테이너 재시작
 - docker restart nginx
5) crond 프로세스 정상 구동중인지 확인하기
 - ps -ef|grep crond
6) 1분마다 프로세스 확인해서 띄워주는 와쳐스케줄 추가 "crontab -e"
 - "* * * * * /data/www/r.d/predis_watcher.sh"
